# PlantUML Diagrams

This directory contains PlantUML sources describing the current `Pretty_LLDB`
architecture.

- `project_architecture_overview.puml`
  High-level layered view of the package architecture.
- `project_module_dependency_map.puml`
  More detailed module-level dependency map for the main Python files.
- `extraction_domain_model.puml`
  Class diagram for the normalized extraction models.
- `tree_summary_sequence.puml`
  Sequence diagram for the tree summary pipeline.
- `web_visualizer_sequence.puml`
  Sequence diagram for the interactive web visualizer command flow.
